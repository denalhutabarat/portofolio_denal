# 🎮 Portofolio - Denal
[![Kunjungi Website](https://img.shields.io/badge/Kunjungi%20Website-Click%20Here-orange?style=for-the-badge)](https://denalhutabarat.vercel.app)

Website portofolio interaktif yang berisi mini game berbasis web dengan sistem leaderboard online menggunakan Supabase.

---

## 🚀 Fitur Utama

- 🎮 2 Game:
  - Snake Game
  - Block Game (Tetris-like)
- 🏆 Leaderboard Online (Top 10)
- 🔄 Update skor otomatis (tidak duplikat)
- 👤 Input nama player (tersimpan di localStorage)
- ⚡ Realtime refresh leaderboard
- 📱 Responsive (bisa dimainkan di HP & PC)

---

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API
- Supabase (Database & API)

---

## 📸 Tampilan

### 🎯 Menu Game
Pilih game yang ingin dimainkan

### 🐍 Snake Game
- Kontrol arah
- Skor bertambah saat makan

### 🧱 Block Game
- Susun block
- Clear line untuk skor

### 🏆 Leaderboard
- Menampilkan Top 10 pemain
- Tidak ada nama duplikat
- Skor otomatis update jika lebih tinggi

---

## 🧠 Struktur Database

Table: `users`

| Column        | Type |
|--------------|------|
| id           | int  |
| player_name  | text |
| score        | int  |
| game_type    | text |
